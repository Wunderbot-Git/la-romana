## 📞 Wenn was kaputt ist (T-1 + Tag 1)

**Production API antwortet nicht:**
```bash
# Logs:
gcloud run services logs read la-romana-api --region us-central1 --limit 100
# Manuell re-deploy:
gcloud builds submit --config cloudbuild.yaml
```

**Migrations re-runnen lassen:**
API-Container restart in Cloud Run-Console (`postgres-migrations` läuft auto beim Boot, idempotent).

**Roll-Back auf vorherigen Build:**
Cloud Run Console → Service → Revisions → "Manage Traffic" → 100% auf vorherige Revision.
